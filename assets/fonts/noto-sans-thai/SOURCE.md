# Noto Sans Thai PDF Assets

The source font is `NotoSansThai[wdth,wght].ttf` from the official Google Fonts repository:

https://github.com/google/fonts/tree/main/ofl/notosansthai

Static server-only PDF assets were instantiated from that official variable TTF with FontTools 4.59.0:

- `NotoSansThai-Regular.ttf`: `wght=400`, `wdth=100`
- `NotoSansThai-Bold.ttf`: `wght=700`, `wdth=100`

The source file SHA-256 is `5a1c559bb539583c8a1fd99d1c5b9491e5e14478c9cd2bd0970d5c3096cc9ef8`.
